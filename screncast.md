# screencast gnome network displays

install:

```bash
sudo apt-get update
sudo apt-get install gnome-network-displays
```

plugin

```bash
sudo apt-get update
sudo apt-get install \
gstreamer1.0-libav \
gstreamer1.0-plugins-good \
gstreamer1.0-plugins-bad \
gstreamer1.0-plugins-ugly
```

firewall sementara nonaktifkan:

```bash
sudo ufw disable
```
